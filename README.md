# Space Zoo Adoption Database

## Overview

Space Zoo Adoption Database is a MySQL relational database project designed for managing an interplanetary creature adoption system.

The database tracks adopters, creatures, species, planets, galaxies, shelters, employees, and adoption records.

## Technologies Used

- MySQL
- MySQL Workbench
- SQL
- Database Design

## Features

- Designed Entity Relationship Diagram (ERD)
- Created normalized relational database structure
- Implemented primary and foreign key relationships
- Applied Third Normal Form (3NF)
- Performed CRUD operations
- Created SQL queries using INNER JOIN, LEFT JOIN, and RIGHT JOIN

## Database Tables

- Adopters
- Adoptions
- Creatures
- Species
- Planets
- Galaxies
- Shelters
- Employees
- Locations
- Zip Codes
